# omochikaeri

這是一個使用 Spring Boot 构建的簡單後端 API，用於管理餐廳訂單。

## 功能

- 管理餐廳菜單項目（新增、更新、刪除）
- 下單並查看訂單歷史
- 基本的管理員和顧客會員資料管理
- RESTful API
- 
## 使用技術

- **Java 11**
- **Spring Boot 2.x**
- **MyBaits**（用於與資料庫交互）
- **PostgreSql 資料庫**（資料庫）
- **Lombok**（減少樣板代碼）
- **Maven**（用於依賴管理）
- **Amazon S3**（用於資料儲存）

## 需求

- Java 11 或更新版本
- Maven 2.x
- PostgreSql 16
